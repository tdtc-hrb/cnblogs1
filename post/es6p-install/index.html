Elasticsearch 6 plus(6, 7) 安装

Zipkin 2.14 adds storage throttling and Elasticsearch 7 support
Feb 19, 2023

Ubuntu 20.04.1

参考“Elasticsearch 2.4 安装”

  1. down 查看版本列表 手动输入版本号
curl -L -O https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-7.17.9-linux-x86_64.tar.gz

config

bootstrap

bootstrap.system_call_filter: false

其中

bootstrap.seccomp: false

在高版本已不可用。

discovery

single-node:

discovery.type: single-node

Ui

git clone https://github.com/mobz/elasticsearch-head.git

install nodejs

cd elasticsearch-head
yarn
yarn run start

open http://localhost:9100/
