cygwin

  • x86

32-bit

Q: How can I install the last Cygwin version for an old, unsupported Windows?
A: Run setup with the options --allow-unsupported-windows --site circa_URL:

tsinghua mirror:

$ setup-x86.exe --allow-unsupported-windows --site https://mirrors4.tuna.tsinghua.edu.cn/sourceware/cygwin-archive/20221123

msys2

  • x86
  • x64

  1. Run mingw32.exe or mingw64.exe from the MSYS2 installation directory (e.g. C:\msys64), depending on whether you want 32-bit or 64-bit Program.

  2. Update package mirrors (may be needed if you have a fresh install of MSYS2)

pacman -Sy pacman-mirrors

x64

pacman -S make gdb \
          mingw-w64-x86_64-gcc

x86

pacman -S make gdb \
          mingw-w64-i686-gcc
