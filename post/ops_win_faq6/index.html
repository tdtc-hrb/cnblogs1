  • exec name
    在主要维护周期, 名字是:
x_community

在次要维护周期, 名字是:

x_Professional
  • target directory
--layout TARGET(directory)
  • components
--add NAME(Component)
  • language
--lang en-US

format

  • Mainstream
.\vs_community.exe --layout D:\win10soft\vs2019 `
--add Microsoft.VisualStudio.Component.CoreEditor `
      ...
--lang en-US
  • Extended
.\vs_Professional.exe --layout D:\win10soft\vs2017 `
--add Microsoft.VisualStudio.Component.CoreEditor `
      ...
--lang en-US

所有的行尾 + 空格 +

 `

尾行除外。
